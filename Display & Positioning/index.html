Display & Positioning

Sebelum melanjutkan membaca, kamu perlu tahu tentang flow of HTML yaitu alur dari sebuah browser untuk merender document dari atas ke bawah, dan kiri ke bawah.

Position Property

Block Element pada asalnya mengambil ruang garis mereka sendiri dan tidak saling menindih dan selalu berawal dari kiri.

Property Value Fungsi
Position Static default value
Relative memposisikan sebuah elemen relatif kepada default position-nya
Absolute semua elemen akan meng-ignore elemen dengan position absolute dan memposisikan elemen relatif terhadap parent element
Fixed menetapkan elemen ke spesifik lokasi di laman web tanpa terpengaruh user-scrolling
Sticky menetapkan elemen tetap pada flow-nya dan akan tetap pada offset position-nya

Offset Properties

Sebelum lanjut lagi, kamu perlu tahu yang namanya Offset Properties yaitu properties yang digunakan untuk memindahkan elemen dari default static position-nya.

  • Top : memindahkan dari atas ke bawah
  • Bottom : memindahkan dari bawah ke atas
  • Right : memindahkan dari kanan ke kiri
  • Left : memindahkan daari kiri ke kanan

Z-Index

(Z-Index) adalah property yang digunakan untuk mengontrol tingkat kedalaman elemen pada web. Dengan value berupa integer, semakin besar integer-nya maka semakin besar elemen tersebut.

Display

Display adalah salah satu property dalam CSS yang berguna untuk mengatur tampilnya suatu elemen.

Value Fungsi
Inline untuk mengatur box element menjadi sesuai ukuran, bisa berbagi tempat dengan elemen lainnya, tetapi width dan height-nya tidak bisa diubah.
Block untuk mengubah box element menjadi line-breakdan mengambil space sendiri, tetapi ukurannya bisa di ubah.
Inline-Block untuk membuat elemen bisa berbagi space dengan yang lain(seperti inline) dan ukurannya bisa diubah(seperti block).
